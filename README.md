# 🏀 CourtKings - Hệ thống đặt sân bóng rổ 24/7

CourtKings là nền tảng quản lý và đăng ký sân bóng rổ hiện đại, tích hợp mã QR để ra vào sân tự động.

## Tính năng chính
- 📊 **Dashboard:** Theo dõi trạng thái sân thời gian thực.
- 📅 **Đặt sân:** Đăng ký ca chơi linh hoạt.
- 🔐 **Mã QR:** Nhận mã QR ngay sau khi đặt để mở cửa sân.
- 📹 **Camera:** Quan sát sân trước khi đến.
- 🛡️ **Admin:** Quản lý sân và người dùng.

## Chạy dự án cục bộ

**Yêu cầu:** Node.js

1. Cài đặt dependencies:
   `npm install`
2. Khởi động server:
   `npm run dev`
3. Truy cập: `http://localhost:3000`
